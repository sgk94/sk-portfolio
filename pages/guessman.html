Guessman

Overview

A fun vanilla JavaScript game where the user must guess the age of the famous singer. Too many wrong guesses and it's game over. Guess while listening to a hit song by the artist for inspiration!

Technology Used

JavaScript | CSS | HTML

Features

  • This game was built using vanilla JavaScript.
  • The goal was to create a fun interactive game where you guess the age of the celebrity.
  • There is also a popular hit song of the artist.
  • There are some various features such as a winning and losing notification and a life counter that ticks down for every wrong answer.
