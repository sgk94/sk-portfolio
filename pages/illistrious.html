Illistrious

Overview

A social app that allows for users to create lists and save links so that friends can enjoy the same posts the user views throughout the week.

Technology Used

JavaScript | Express | Node | CSS | HTML | MongoDB/Mongoose | Materialize | MVC | OAuth

Features

  • A social media app that allows users to create lists and save links for friends to see.
  • This app features a friend system allowing users to add each other and to either accept or decline friend requests.
  • The front end was developed using Express with the MVC model.
  • MongoDB/Mongoose was used for the backend with three schemas: User, List and Friend.
  • The friend model was specifically for holding pending friend requests with information on a recipient and a sender.
  • The app employs O Auth for user log in.
